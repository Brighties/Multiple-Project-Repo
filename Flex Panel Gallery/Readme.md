## Description

This project showcases flex panels, providing an interactive user experience when clicking on the panels. Each panel can be expanded to reveal more content, creating a dynamic and engaging layout.

## Features

- Click on a panel to toggle its size and reveal additional content.
- Smooth transition effects for a visually pleasing experience.
